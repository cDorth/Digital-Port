Integrar um sistema de IA que analisa o projeto que o visitante está visualizando e sugere outros projetos relacionados automaticamente.

Mostrar sugestões como cards no final da página do projeto.

Criar uma linha do tempo animada exibindo projetos e conquistas em ordem cronológica.

5. Comparador de Habilidades

Criar uma funcionalidade onde o visitante escolhe duas skills do portfólio.

O sistema exibe projetos, métricas e resultados que comprovam sua proficiência em cada skill.

Interface amigável e responsiva, com gráficos e indicadores visuais.

Conectar cada projeto ao repositório GitHub correspondente, exibindo commits recentes e link direto.

7. Banco de Dados Integrado – SQLite

Banco de dados único e centralizado em SQLite (database.db) com tabelas para: usuários, projetos, conquistas, curtidas, comentários, notificações, habilidades e integrações externas.

requirements.txt atualizado com todas as bibliotecas usadas no projeto.
